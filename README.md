# 📌 Sistema de Gerenciamento de Usuários (CLI)

Aplicação desenvolvida em **Node.js** para gerenciamento de usuários via terminal (CLI), aplicando boas práticas de arquitetura, padrões de projeto (**GRASP e GOF**) e persistência de dados em arquivo `.txt`.

---

## 🚀 Funcionalidades

- ✅ Cadastro de usuários  
- 📋 Listagem de usuários  
- ✏️ Edição de usuários  
- ❌ Remoção de usuários  
- 💾 Persistência em arquivo (`users.txt`)  
- ✔️ Validação de dados (nome e email)
